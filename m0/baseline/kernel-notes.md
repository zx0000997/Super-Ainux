# Baseline kernel — Linux 6.19.8 (optional custom build)

The repository vendors **Linux 6.19.8** under `linux-6.19.8/` (see [OS_DEVELOPMENT_GUIDE.md](../../OS_DEVELOPMENT_GUIDE.md)).

## Branding / reproducibility

- Run `make menuconfig` or start from `defconfig` / distro config.
- Set **local version** so `uname -r` identifies the SAS baseline, e.g.:

  ```text
  General setup → Local version - append to kernel release
  ```

  Example value: `-sas-m0` → release string like `6.19.8-sas-m0`.

- Save your configuration as a fragment in this repo if you want bit-repro builds, e.g. `m0/baseline/kernel-config.fragment` (not generated by default).

## Relationship to M0a

M0a **does not require** replacing the Debian stock kernel. Use a custom 6.19.8 build when you need driver, hardening, or version alignment with the Super Ainux kernel guide.
